This dataset is for the forex pair EUR-AUD.    
This dataset looks at the **Daily** _Open_, _High_, _Low_, _Close_.   

# Dataset Format  

YYYY-MM-DD    

# Dataset Range    

Daily Data from: 2004-06-17 to 2023-01-10.    

# Additional Parameters    

* Tick_volume    
* Spread    
* Real_Volume
